About This Project

This demo site is part of a User Analytics Application built to demonstrate real-time session tracking, click analytics, and heatmap generation.

Every interaction you make on this page — from scrolling to clicking buttons — is captured by a lightweight tracking script and sent to the analytics backend.

🔍 How Tracking Works

A vanilla JS script captures page views and clicks using sendBeacon API for maximum reliability.

🔒 Privacy First

No personal data is collected. Only anonymous session IDs and click coordinates are recorded.

💾 Data Storage

Events are stored in MongoDB and can be queried via a RESTful API for dashboard visualizations.

📊 Dashboard

A Next.js dashboard provides session timelines and interactive heatmap views of user clicks.
